# Next.js Application

This repository contains a **Next.js** application bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app). It follows the App Router architecture and is ready for local development and deployment.

## Getting Started

Start the development server using your preferred package manager:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Once running, open your browser and navigate to:

```
http://localhost:3000
```

The application will automatically reload as you make changes.

---

## Project Structure

* **`app/page.tsx`** – Main entry page of the application. Edit this file to update the UI.
* **App Router** – Uses the modern Next.js App Router for routing and layouts.
* **Font Optimization** – Built-in font optimization via `next/font`.

This project uses **Geist**, the official Vercel font, which is automatically optimized and loaded.
